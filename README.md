# vimfiles

## Quick Start

### Linux

```
cd %HOME%
git clone https://github.com/dcharb/vimfiles.git .vim
cd .vim
setup.sh
```

Open vim and run `:PluginInstall` to install all the plugins.

### Windows

```
cd %USERPROFILE%
git clone https://github.com/dcharb/vimfiles.git
cd vimfiles
setup.bat
```

Open vim and run `:PluginInstall` to install all the plugins.
